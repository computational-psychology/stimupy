
lengths_from_visual_angles_ppd
==============================



.. py:function:: stimupy.utils.lengths_from_visual_angles_ppd(visual_angles, ppd, round=True)


   Calculate lengths (pixels) from visual angles (degrees) and pixels-per-degree

   :param visual_angles: list of visual angles
   :type visual_angles: Sequence[float, float, ...] or None
   :param ppd: pixels per degree
   :type ppd: int or None
   :param round: if True, round output length to full pixels
   :type round: bool

   :rtype: List with visual angles (degrees) translated to lengths (pixels)
